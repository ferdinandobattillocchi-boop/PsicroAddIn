# PsicroAddIn- 
Add-in Excel per calcoli psicrometrici (C# &amp; C) 
Si trattata di un piccolo addin (file .xll) da caricare in Excel per avere disponibile una libreria dedicata ai calcoli psicrometrici.
Nella cartella release sono presenti i file:
- PsicroAddIn-AddIn64-packed.xll per Excel 64 bit
- PsicroAddIn-AddIn-packed.xll per Excel a 32 bit
I calcoli sono stati effettuati 
## Utilizzo:
Una vola caricato in Excel saranno disponibili le funzioni:

A. PSICRO(p1,v1,p2,v2,target,units)
B. PISCRO.HELP()
C. PSICRO.INFO()
D. PSICRO.UNITS.INFO()
E. PSICRO.SET.QUOTA()

FUNZIONE PSICRO(p1,v1,p2,v2,target,units) è la funzione principale che restituisce una o più variabili psicrometriche
